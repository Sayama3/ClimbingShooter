# Climbing Assassin

## Commands
- ZQSD          (Move / Climbing Movement)
- Space Bar     (Jump / Climb / Unclimb)
- Right Click   (Aiming)
- Left Click    (Shooting)

## Assets Used :

- Third Person Template
- Unreal Engine Starter Pack
- SimpleClimbingAnims
- Infinity Blade Effects
- Anim Starter Pack
- FPS Weapon Bundle
- Some sound from the internet

## Features

- Movement
- Aiming
- Shooting
- Climbings
- IAs
